This is a small project for AI job match.  
It helps user upload resume and see best matching jobs.  

Features
- Upload resume and get job match %.  
- Shows top job cards with match score.  
- AI chat assistant.  

How to use
1. Open `index.html` in browser.  
2. Click "Upload Resume" and select your file.  
3. See job cards generated below.  
4. Click "AI Chat" to ask about scores or roles.  

Notes
- This is just prototype, not full product.  

Tech
- HTML, CSS, Javascript only.  
- Frontend only, no backend required.
